
# Stock Return Calculator

## Overview

This application calculates the return on a stock investment.

## Features

*   Users can input the purchase price, selling price, and number of shares.
*   The application calculates the total profit or loss.
*   The application calculates the return on investment (ROI) as a percentage.
*   The user interface is styled for a clean and modern look.
